📊 Data Analysis & Business Insights
📌 Project Overview

This project analyzes business data to understand the effect of a new website design on customer conversions and revenue. By using data analysis, visualization, and statistical testing, the project helps identify whether the new website performs better than the old version.

🎯 Objective
Compare the performance of the Old Website and New Website
Analyze customer conversion rates and revenue
Use statistical testing to validate the results
🛠️ Technologies Used
Python 🐍
Pandas
Matplotlib
SciPy
📂 Dataset Features

The dataset contains the following columns:

customer_id – Unique ID for each customer
month – Month of the data record
website_version – Old or New website version
visits – Number of visits by customers
conversions – Number of successful conversions
revenue – Revenue generated from customers
⚙️ Project Process
Load and preprocess the dataset
Create a new feature called Conversion Rate
Analyze the data to identify patterns and trends
Perform Hypothesis Testing (T-test) to check statistical significance
Create visualizations to present the results
Save important insights to a file
📊 Key Insights
The new website version shows a higher conversion rate
Revenue generated from the new website is greater
Customer engagement has improved compared to the old version
📈 Hypothesis Testing
Null Hypothesis (H₀): There is no difference in conversion rates between the old and new website.
Alternative Hypothesis (H₁): The new website improves conversion rates.

Result:
If P-value < 0.05, the result is statistically significant.
This indicates that the new website performs better than the old website.

📉 Visualizations
Bar chart comparing conversion rates
Bar chart comparing revenue
📁 Output
business_insights.txt – Contains the analysis results and important findings.
🚀 Conclusion

The analysis shows that the new website design improves both conversion rates and revenue. Therefore, adopting the new website version can help businesses achieve better performance and customer engagement.
