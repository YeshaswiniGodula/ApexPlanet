# Apex Data Analysis Project

This repository contains different tasks related to data analysis and customer segmentation using Python. The project focuses on analyzing datasets, calculating key performance metrics, and visualizing insights using Python libraries.

## Project Description
The goal of this project is to perform data analysis and segmentation to understand patterns in the dataset. By using different analysis techniques, the project helps in identifying trends and meaningful insights from the data.

## Tasks Included

### Task 1 – Data Processing
In this task, the dataset is cleaned and preprocessed. Missing values are handled and the data is prepared for further analysis.

### Task 2 – Segmentation Analysis
This task focuses on customer segmentation using the given dataset. The data is analyzed to group similar records and understand patterns in the data.

### Task 3 – Data Visualization
In this task, charts and graphs are created to visualize the analysis results. Visualization helps in understanding the data more clearly.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Files Included
- task1 – Data preprocessing scripts  
- task2 – Segmentation analysis code  
- task3 – Visualization and analysis code  
- CSV files – Dataset used for analysis  

## Conclusion
This project demonstrates how Python can be used for data analysis, segmentation, and visualization to extract useful insights from datasets.
